# kloger
herramienta keylogger para el curso de hacking etico


## Características
Ejecución en segundo plano mediante extensión `.pyw`.
Captura de texto legible con marcas de tiempo (Timestamps).
para detener el proceso del keylogger se digita lo siguiente (`Ctrl + Alt + S`).

## 🛠️ Instalación y Uso
instalar la siguiente libreria de esta manera:
pip install pynput
ejecutar en modo consola: `python logger.py`

  ejecutar en segundo plano: `pythonlogger.pyw`
